付属CD-ROMについて

この付属CD-ROMには、本書で紹介したサンプルプログラムの
ソースコード（.javaファイル）と、
それらをコンパイルした結果の.classファイルを収録しています。

ご利用になる前に、次の注意事項に留意しておいてください。

■　CD-ROMは、原則として、破損しても交換できません。
　　必要に応じてバックアップをとって大切に保管してください。

■　サンプルプログラムを使用、運用した結果については、
　　著者、訳者、編集者、株式会社ピアソン・エデュケーションは
　　その責任を一切負いません。
　　どのような場合も使用者の責任において使用してください。
